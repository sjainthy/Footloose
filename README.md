# Footloose

**Footloose** is a modern footwear brand concept focused on stylish, comfortable, and affordable shoes for everyday movement. The brand is designed for people who want footwear that looks good, feels light, and supports their daily lifestyle — from work and travel to casual outings and weekend plans.

## About the Company

Footloose aims to provide footwear that combines comfort, durability, and modern design. The brand focuses on practical designs suitable for daily Indian lifestyles, offering shoes that can be worn for long hours without compromising on appearance.

The company’s core idea is simple: **step freely, move confidently, and stay comfortable all day.**

## Brand Focus

Footloose focuses on:

* Comfortable everyday footwear
* Lightweight and breathable shoe designs
* Durable soles for regular use
* Stylish collections for casual and smart-casual wear
* Affordable footwear options for a wide customer base

## Product Categories

The Footloose website highlights three main footwear categories:

### Urban Sneakers

Clean and lightweight sneakers designed for daily use, travel, college, office, and casual outings.

### Comfort Walkers

Soft, cushioned walking shoes made for long hours, regular walking, and daily comfort.

### Smart Casuals

Stylish semi-formal footwear that pairs well with denim, chinos, and smart-casual outfits.

## How to Use

1. Create a GitHub repository named **footloose**.
2. Add the `index.html` file to the repository.
3. Add this `README.md` file to describe the project.
4. Commit and push the files to GitHub.
5. Enable GitHub Pages from the repository settings.

## GitHub Pages Deployment

To publish the website using GitHub Pages:

1. Open the repository on GitHub.
2. Go to **Settings**.
3. Click **Pages**.
4. Under **Build and deployment**, select **Deploy from a branch**.
5. Choose the **main** branch and `/root` folder.
6. Click **Save**.

After deployment, the website will be available at:

```text
https://your-github-username.github.io/footloose/
```
